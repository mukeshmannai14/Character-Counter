# Character Counter

A simple and responsive **Character Counter** application built using **HTML, CSS, and JavaScript**. The application tracks user input in real time, displays the number of characters typed, shows the remaining character count, and prevents users from exceeding the maximum character limit.

## 🚀 Features

* Real-time character counting
* Displays typed and remaining characters
* Maximum character limit (200 characters)
* Prevents users from typing beyond the limit
* Warning message when the character limit is reached
* Clean and responsive user interface

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (DOM Manipulation & Event Listeners)

## 📖 How It Works

1. The user types a message into the text area.
2. JavaScript listens for the `input` event.
3. The application updates the character count in real time.
4. It displays the number of characters typed and the remaining characters.
5. When the maximum limit of 200 characters is reached, the application prevents additional input and displays a warning message.
